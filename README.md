# CNC Torno Pro v5

Calculadora profesional offline para torno CNC - Metalmecánica Aluminio & Hierro

## Características
- Cálculo de velocidades, avances, tiempos, potencia real con eficiencia
- Módulo de roscado completo
- Biblioteca ISO completa: CNMG, VNMG, WNMG, DNMG, TNMG, CNGA, PCD
- 10 máquinas: Haas ST-10/20/30/DS-30Y, Doosan Lynx/Puma, Mazak, Okuma, DMG Mori
- 8 herramientas, 11 refrigerantes
- Presets con localStorage
- Tema claro/oscuro
- 100% responsive móvil
- Funciona sin internet
